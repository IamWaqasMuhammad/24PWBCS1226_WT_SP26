# 24PWBCS1226_WT_SP26 - Web Technologies Learning Repository

This repository contains all my coursework and learning materials for the university subject **24PWBCS1226 Web Technologies (WT) SP26**. It covers web development technologies including HTML, CSS, JavaScript, React, Express.js, and Node.js.

## 📚 Course Overview

This subject focuses on building modern web applications, starting with fundamental web technologies and progressing to full-stack development.

### Technologies Covered

- **HTML5** - Web structure and markup
- **CSS3** - Styling and responsive design
- **JavaScript (ES6+)** - Client-side programming
- **React** - UI development
- **Express.js** - Backend framework
- **Node.js** - Server-side runtime

## 🚀 Getting Started

### Prerequisites

- Modern web browser
- Code editor (VS Code recommended)

### Running Projects

1. Clone the repository
2. Open HTML files directly in your browser
3. For future Node.js/React projects: `npm install && npm start`
